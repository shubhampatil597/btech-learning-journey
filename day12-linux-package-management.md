# Day 12 – Linux Package Management & Software Installation

## What is Package?

Package is software bundle containing:
- program files
- libraries
- configuration files

Examples:
- git
- nano
- python

---

## Package Manager

Package manager installs and manages software.

Examples:
- apt
- pkg
- dnf
- pacman

---

## Repository

Repository is trusted online software storage.

Linux downloads packages from repositories.

---

## Important Commands

### Update package list
pkg update

### Upgrade packages
pkg upgrade

### Install package
pkg install nano

### Search package
pkg search python

### List installed packages
pkg list-installed

### Remove package
pkg uninstall nano

---

## Dependencies

Dependencies are software required by another software.

Linux installs them automatically.

---

## Key Learnings

- Linux software uses packages
- Package managers automate installation
- Repositories provide trusted software
- Dependencies are managed automatically
