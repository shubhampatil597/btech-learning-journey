# Day 1 – Computer Basics 

## What is a Computer?

A computer is an electronic machine that:
- Takes input
- Processes data
- Gives output
- Stores information

Flow:
Input → Process → Output → Storage

---

## Input Devices
- Keyboard
- Mouse
- Microphone
- Touchscreen

---

## Output Devices
- Monitor
- Speaker
- Printer

---

## Storage Devices
- Hard Disk (HDD)
- Solid State Drive (SSD)
- Pen Drive

---

## Types of Computers
- Desktop
- Laptop
- Smartphone
- Server
- Supercomputer

---

## My Understanding

A computer is not just a laptop.  
Even a mobile phone is a computer because it takes input, processes it, and gives output.

---

## Key Learnings
- Input and output are different
- CPU does processing
- Storage saves data
