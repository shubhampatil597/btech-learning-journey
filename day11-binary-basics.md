# Day 11 – Binary Basics & Data Representation

## 1. Binary System

Computers use binary system based on:
- 0
- 1

because electronics work using ON/OFF electrical states.

---

## 2. Bit

Bit = Binary Digit

Single 0 or 1 is called bit.

---

## 3. Byte

8 bits together form 1 byte.

Example:
10101010

---

## 4. Storage Units

- 1 Byte = 8 bits
- 1 KB = 1024 Bytes
- 1 MB = 1024 KB
- 1 GB = 1024 MB

---

## 5. Decimal vs Binary

Humans use decimal system (0–9).

Computers use binary system (0 and 1).

---

## 6. Decimal to Binary Example

Decimal 10:
1010

Decimal 5:
101

---

## 7. Binary to Decimal Example

1011:
8 + 2 + 1 = 11

---

## 8. Data Representation

Text, images, videos and applications are internally stored as binary.

---

## 9. Key Learnings

- Computers use binary
- Binary uses 0 and 1
- Bit is smallest unit
- Byte = 8 bits
- All digital data becomes binary
