# Day 10 – Linux Networking Basics & Internet Fundamentals

## 1. Networking

Networking means communication between devices.

Examples:
- internet
- WiFi
- cloud servers

---

## 2. IP Address

Every network device gets unique IP address.

Example:
192.168.1.10

IP identifies device on network.

---

## 3. DNS

DNS converts domain names into IP addresses.

Example:
google.com → IP address

---

## 4. Ports

Ports identify services/applications.

Examples:
- HTTP → 80
- HTTPS → 443
- SSH → 22

---

## 5. HTTP vs HTTPS

HTTP:
- normal website communication

HTTPS:
- secure encrypted communication

---

## 6. Client-Server Model

Client sends request.
Server sends response.

Example:
Browser → Server

---

## 7. Important Commands

### ping
Checks connectivity.

Example:
ping google.com

---

### ip a
Displays IP information.

---

### curl
Fetches data from server.

Example:
curl google.com

---

### hostname
Displays system hostname.

---

## 8. Internet Request Flow

Browser → DNS → Server → Response

---

## 9. Key Learnings

- Internet depends on networking
- IP identifies devices
- DNS converts names to IPs
- Ports identify services
- Client-server model powers websites
