# Day 4 – Operating System Basics

## 1. What is OS?

Operating System is software that manages hardware and allows users to interact with the computer.

---

## 2. Functions of OS

- Memory management
- Process management
- File management
- Device management
- User interface

---

## 3. File System

File system organizes data into files and folders.

Example:
Documents/
Projects/
Downloads/
