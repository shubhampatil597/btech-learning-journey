# Day 6 – Linux Command Line & Navigation

## 1. Linux Working

User → Shell → Kernel → Hardware

Shell interprets commands and kernel executes them.

---

## 2. Bash

Bash is the default shell used in Linux systems.

---

## 3. Command Structure

command [options] [arguments]

Example:
ls -l /home

---

## 4. Navigation Commands

pwd → current directory  

ls → list files  
ls -l → detailed  
ls -a → hidden  

cd → change directory  
cd .. → go back  
cd ~ → home  
cd / → root  

---

## 5. File & Folder Commands

mkdir → create folder  
mkdir -p → nested folders  

touch → create file  

rm → delete file  
rm -r → delete folder  

---

## 6. Path Types

Absolute Path:
/home/user/docs

Relative Path:
docs/file.txt

---

## 7. Hidden Files

Files starting with "." are hidden

Use:
ls -a

---

## 8. Key Learnings

- Linux uses command interface
- Navigation is core skill
- Paths are important
- Commands follow structure

## What is Linux?

Linux is an operating system that manages hardware and runs software.

It is open-source and widely used in servers and cloud systems.

## Why Linux is Used

- Used in servers and cloud
- Fast and lightweight
- Secure and stable
- Provides full system control
- Free and open-source
