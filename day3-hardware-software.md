# Day 3 – Computer Fundamentals

## 1. Motherboard

The motherboard is the main circuit board of a computer that connects all components like CPU, RAM, storage, and GPU.

It acts like a communication system between all hardware parts.

---

## 2. GPU (Graphics Processing Unit)

GPU is responsible for handling graphics and visual output.

Uses:
- Gaming
- Video editing
- Rendering
- AI tasks

CPU = general tasks  
GPU = parallel heavy tasks

---

## 3. Hardware vs Software

### Hardware
Physical components of computer:
- Keyboard
- Mouse
- CPU
- RAM
- SSD

### Software
Programs or instructions:
- Operating systems
- Applications
- Browsers

Example OS:
- Windows
- Android
- Linux

---

## 4. How Computer Works (Example: Opening App)

1. User gives input (click)
2. OS receives command
3. CPU processes it
4. RAM loads active data
5. Storage provides files
6. GPU displays output
7. Motherboard connects all parts
