# Day 13 – Linux Text Editors (nano & vim)

## Why Text Editors Important?

Linux uses text configuration files for:
- server settings
- scripts
- logs
- configurations

Terminal editors are important in Linux administration.

---

## nano Editor

nano is beginner-friendly terminal text editor.

Open file:
nano notes.txt

Save:
CTRL + O

Exit:
CTRL + X

---

## Useful nano Shortcuts

- CTRL + O → Save
- CTRL + X → Exit
- CTRL + K → Cut line
- CTRL + U → Paste
- CTRL + W → Search

---

## vim Editor

vim is advanced terminal text editor.

Used heavily by Linux administrators and DevOps engineers.

Open file:
vim demo.txt

---

## Vim Modes

### Normal Mode
Used for commands/navigation.

### Insert Mode
Used for typing text.

Press:
i

to enter Insert Mode.

Press:
ESC

to exit Insert Mode.

---

## Save & Exit vim

Save and quit:
:wq

Quit without saving:
:q!

---

## Key Learnings

- Linux servers use terminal editors
- nano is beginner-friendly
- vim is advanced and powerful
- Linux configs are mostly text files
